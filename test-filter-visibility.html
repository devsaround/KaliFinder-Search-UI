Test 2: Partial Facet Configuration

This test verifies that only configured facets are visible. Configure some facets in DashboardVendorFacets and verify only those show up.

Steps to Test:

  1. Go to DashboardVendorFacets in the admin panel
  2. Add and configure only "Brand" and "Color" facets
  3. Set their visibility to true
  4. Save the configuration
  5. Go back to the search page
  6. Check the console logs for the facet configuration
  7. Verify only Brand and Color filters are visible
Expected Result: Only Brand and Color filter sections should be visible. All other filters (Category, Price, Size, Tags, Stock Status, etc.) should be hidden.

Test 3: Mobile Filter Visibility

This test verifies that the mobile filter drawer also respects the facet configuration.

Steps to Test:

  1. Open the search page on a mobile device or use browser dev tools mobile view
  2. Click the filter button (hamburger menu or filter icon)
  3. Check that the mobile drawer only shows configured filters
  4. Verify the accordion doesn't have any sections expanded by default
Expected Result: The mobile filter drawer should only show the configured filter sections, and none should be expanded by default.

Test 4: Accordion Default Behavior

This test verifies that the accordion doesn't expand all sections by default.

Steps to Test:

  1. Configure some facets in DashboardVendorFacets
  2. Go to the search page
  3. Check that the filter accordion sections are collapsed by default
  4. Verify that only the configured sections are visible, but collapsed
Expected Result: All accordion sections should be collapsed by default. Users should need to click to expand them.
